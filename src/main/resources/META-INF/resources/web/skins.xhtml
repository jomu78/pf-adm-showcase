Skins Switch the current showcase session between the AdminFaces-compatible `pf-adm` skins.

Choose a skin and the page will reload immediately so the header, sidebar, and shared surface tokens pick up the new template-level skin classes.

Current skin #{skinView.skin}
Default skin #{showcaseLayoutView.defaultSkin}
Scope Session-local via skinView

Navbar

Uses navbar-#{skinView.skin} from the template shell.

Sidebar

The sidebar and brand block switch with the same session value.

Compatibility

Only values from SkinEnum are accepted by the setter.
